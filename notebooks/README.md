# MIT_OCW_Linear_Algebra_18_06

I contacted Prof Gilbert Strang in 2018 to ask if I could write content for his MIT Open Courseware course on linear algebra (MIT 18.06) using the Python language. At the time of the recording of these famous lectures, he was using MATLAB code. He graciously agreed.

This repository contains Jupyter notebooks that I created to accompany the lectures in the course. The notebooks include code examples and explanations of the concepts covered in the lectures.

I am currently updating the notebooks.
